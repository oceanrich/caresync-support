常见问题

🚀 如何开始使用CareSync？

1. 下载并安装CareSync应用

2. 首次打开时，应用会智能检测您的系统语言

3. 点击"添加用药提醒"开始添加您的第一个药物

4. 设置用药时间和频率，保存即可开始接收智能提醒

📸 如何使用拍照识别功能？

1. 在添加药物时选择"拍照识别"选项

2. 对准药盒或说明书上的文字信息拍照

3. 应用会自动识别药物名称、剂量等信息

4. 确认信息无误后保存即可

注意：免费版每月可使用10次，Pro版无限制

🔒 我的数据安全吗？

绝对安全！CareSync采用完全本地存储策略：

• 所有用药数据存储在您的设备本地

• OCR识别完全在本地处理，不上传任何信息

• 不需要注册账号或联网同步

• 您的健康隐私得到最高级别保护

🌍 支持哪些语言？

CareSync支持6种主要语言：

• 简体中文 • 繁体中文 • 英语

• 日语 • 韩语 • 西班牙语

应用会智能检测您的系统语言并自动切换，也可以在设置中手动选择。

💳 如何升级到Pro版？

1. 在应用内点击"升级到Pro"

2. 选择月订阅或年订阅计划

3. 通过App Store完成购买

4. 立即享受无限药物管理和拍照识别功能

🤖 AI智能提醒是如何工作的？

CareSync的AI系统会学习您的用药习惯：

• 分析您的服药时间模式

• 预测最佳提醒时间

• 提供个性化用药建议

• 评估依从性风险并及时提醒
